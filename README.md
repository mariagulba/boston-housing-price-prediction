# Boston Housing Price Prediction

Projekt analizy danych i predykcji cen nieruchomości na podstawie zbioru Boston Housing.

## Technologie
Python
Pandas
Scikit-learn
Matplotlib

## Modele
- Linear Regression
- Random Forest
- Ridge
- Lasso

## Cel
Porównanie skuteczności modeli w przewidywaniu cen nieruchomości.
